# hz-portfolio

Personal site + blog. Static HTML, generated by a small Node script, hosted
on GitHub Pages.

## Stack

- **`build.js`** — the entire generator (markdown → HTML)
- **`marked`** + **`gray-matter`** + **`highlight.js`** (only npm deps)
- **Tailwind** + **Alpine.js** via CDN (no build step for CSS or JS)
- **`instant.page`** for prefetch-on-hover (links feel instant)
- **GitHub Actions** deploys `public/` to GitHub Pages on push to `main`

## Layout

```
content/
  home.md                 -> /
  about.md                -> /about/
  posts/
    <slug>/
      index.md            -> /posts/<slug>/
      <any-image>.png     pasted images live next to the post
static/                   site-wide assets (favicon, /images/*)
build.js                  generator
public/                   build output (gitignored)
```

## Day-to-day

```bash
# write a new post
make new-post SLUG=my-new-post
$EDITOR content/posts/my-new-post/index.md
# paste images into content/posts/my-new-post/, reference as ![alt](pic.png)

# preview locally
make serve            # http://localhost:8000

# ship it
git add content/posts/my-new-post
git commit -m "post: my new post"
git push              # GitHub Action deploys in ~30s
```

## Commands

- `make install` — install npm deps
- `make build` — build into `public/`
- `make serve` — build + serve at http://localhost:8000
- `make new-post SLUG=<slug>` — scaffold a new post folder + frontmatter
- `make clean` — wipe `public/`

## Frontmatter

```yaml
---
title: "Title of the post"
date: 2026-05-17
summary: "Optional one-liner shown later in lists"
tags: [data, tools]
draft: false        # set true to hide from lists/build
---
```
